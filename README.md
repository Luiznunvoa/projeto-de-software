# Para rodar

```bash
# instala as dependências
bun install

# roda
bun run dev
```

# Se você não tem o bun instalado...

## Instala assim no linux

```bash
curl -fsSL https://bun.com/install | bash
```

## E assim no windows

```bash
powershell -c "irm bun.sh/install.ps1|iex"
```
